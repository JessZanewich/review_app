RESTful Routing

A table of the 7 REStful routes

Name Path HTTP Purpose Mongoose Method
Index /reviews GET List all Reviews Review.find()
New /reviews/new GET New review form N/A
Create /reviews POST Create a new review, then redirect Review.create()
Show /reviews/:id GET Show specific review Review.findById()
Edit /reviews/:id/edit GET Edit form for a specfic review Review.findById()
Update /reviews/:id PUT Update review, then redirect Review.findByIdAndUpdate()
Destroy /reviews/:id DELETE Delete a review, then redirect Review.findByIdAndRemove()
